# 🚀 macOS Minecraft Server CLI

A zero-touch, self-updating bash script designed to host a secure Minecraft Java
server on a Mac mini (or any modern macOS machine). No port forwarding required,
zero manual file managing, and it is entirely script-managed.

## ✨ Features

- **Zero-Touch Install:** Fetches the absolute latest server jar from Mojang
  automatically.
- **No Port Forwarding:** Uses `playit.gg` to securely share your server with
  friends without exposing your home IP address or router.
- **Automatic Updates:** Checks Mojang's API every 24 hours. If an update is
  live, it stops the server, moves the files, restores the world, and boots back
  up!
- **Isolated Security:** Automatically creates a dedicated, hidden macOS user
  (`mcserver`) to isolate the server software from the rest of your computer.
- **Intelligent Backups:** Every update pushes the previous environment into an
  organized archive folder so you never lose your progress.
- **Native CLI Feel:** Adds a system alias so you can manage your server just by
  typing `mc`.
- **Seamless Migration:** Prompts you during the initial setup to drag and drop
  a pre-existing server folder you want to start from.

## 📥 Quick Start & Installation

To kick off the initialization, open your Mac's **Terminal** and run the
following combined command:

```bash
curl -fL -o mc https://raw.githubusercontent.com/chaystonwood/macos-minecraft-server-tooling/master/mc && chmod +x mc && ./mc
```

You don't even need to clone this repository!

> ⚠️ _Note: The script will ask for your administrator password during the
> initialization to create the isolated user account and verify your firewall._

### 🔗 Finalizing Playit.gg Setup

Because the Playit agent runs as a plugin directly inside your Minecraft server,
it will generate an authentication link the very first time you boot the server.

1. **Restart your terminal** (or open a new window) so the `mc` shortcut
   activates.

2. Start your server by running:

```bash
mc start
```

3. Enter the live server console to see the plugin's output:

```bash
mc console
```

4. Scroll up in the logs or wait a few seconds until you see a generated URL
   message like: `To claim your agent visit: https://playit.gg/claim/...`

5. Copy that link into your web browser go to the Playit.gg website.

6. Add a tunnel designated for **Minecraft Java**.

7. Once the website says your tunnel is "Active", use the keyboard shortcut
   `Ctrl + B` and then tap `D` to safely detach from the console without turning
   the server off.

> 💡 Warning: Do not use `Ctrl + C` to exit the session, as doing so will
> terminate the Minecraft server process!

## 🌐 How to Join & Find Your Server IP

Because this script uses Playit.gg, your home IP address remains completely
hidden and protected. Your friends will connect using a secure, custom domain
generated by Playit.

1.  Go to [Playit.gg](https://playit.gg/login) and log in.
2.  Click on the **Tunnels** tab.
3.  Look for your **Minecraft Java** tunnel.
4.  You will see an address that looks something like
    `random-words.gl.at.ply.gg:12345`.

**Give that exact address to your friends! They can paste it directly into their
Minecraft "Direct Connection" or "Add Server" screen.**

> 💡 **For the Host:** If you are playing Minecraft on the _same_ Mac that is
> running the server, you don't need to use the Playit address. You can simply
> connect using the IP: `localhost`

## 🕹️ CLI Usage Commands

After running the quick start, handle the day-to-day commands directly through
the CLI shortcut.

| Command        | Action                                                                |
| :------------- | :-------------------------------------------------------------------- |
| `mc start`     | Starts the Minecraft server safely in a background tmux session.      |
| `mc stop`      | Gracefully sends the stop command to save your world and shuts down.  |
| `mc console`   | Attaches to the live server console to run commands.                  |
| `mc status`    | Shows if the server is currently running or stopped.                  |
| `mc update`    | Force a check against the Mojang API and pull an update if necessary. |
| `mc uninstall` | Permanently stops, unloads the scheduler, and wipes the directory.    |

> 💡 **Pro-Tip:** To safely leave the session after unning `mc console` (without
> stopping the server), press `Ctrl + B`, then `D`.

## 🚨 Uninstallation

If you ever want to move your server off the Mac or simply want to clean up your
machine, run the following command. It stops active environments, removes the
automated update schedule from launchd, deletes the hidden user, and cleans the
target folder.

```bash
mc uninstall
```

> _🛡️ **Note:** To keep your progress safe, this command will automatically pack
> your entire application directory (including active worlds, plugins, and
> update archives) into a dated ZIP file on your Mac's Desktop before deleting
> the local environment!_
